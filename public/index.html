Setup Free Google Sheets Sync

Step 1: Connect Meta to Google Sheets

Use Meta's Native Sync (100% Free)

Facebook allows you to export Lead Ads directly to Google Sheets for free. This bypasses the need for complex API apps or paid Zapier accounts.

  1. Go to your Facebook Page > Lead Center or publishing tools.
  2. Find your Lead Ad form and click "Connect CRM".
  3. Search for Google Sheets and follow the prompts to create a new sheet.

Once your leads are syncing to that Google Sheet, move to the next step.

Add the Bridge Script

Now, let's connect that Google Sheet to your Simple CRM. Open the Google Sheet where leads are flowing.

  1. In the menu bar, click Extensions > Apps Script.
  2. Delete the default code and paste the script below.
  3. Hit the "Save" (disk) icon.
function onFormSubmit(e) {
  var sheet = SpreadsheetApp.getActiveSpreadsheet().getActiveSheet();
  var row = sheet.getLastRow();
  var data = sheet.getRange(row, 1, 1, sheet.getLastColumn()).getValues()[0];
  
  // Adjust these indexes (0,1,2) based on which columns your Name, Email, and Phone are in.
  var payload = {
    "name": data[1], // Example: Column B is Name
    "email": data[2], // Example: Column C is Email
    "phone": data[3], // Example: Column D is Phone
    "source": "Meta FB Ads (Free Sync)"
  };

  var options = {
    "method": "post",
    "contentType": "application/json",
    "payload": JSON.stringify(payload)
  };

  // Replace this URL with your active Ngrok generic webhook
  UrlFetchApp.fetch("YOUR_NGROK_URL/api/webhooks/generic", options);
}

Set the Trigger & You're Done!

Finally, we need to tell Google Sheets to run this script automatically when a new lead arrives.

  1. In the Apps Script editor, click on the Clock Icon (Triggers) on the left sidebar.
  2. Click + Add Trigger in the bottom right.
  3. Settings: Choose onFormSubmit, Head, From Spreadsheet, On change or On form submit.
  4. Save it. (Google will ask you to authorize the script).

That's it! 100% free, instant lead syncing without any third-party fees.
